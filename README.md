# Shoreline CS Club

This repository contains the official codebase for the **Shoreline Computer Science Club** online platform.

The project is being developed in phases and will evolve into a full platform that includes:
- Educational courses
- Community tools for students
- A club creation and management system
- Security and technical learning resources

## Planned Scope

The long-term vision for this project includes:

- A public-facing website for the Shoreline CS Club  
- Beginner and advanced programming courses (Python, Linux, Django, and more)  
- A platform that allows students to create and manage their own clubs  
- Educational content focused on security and safe computing practices  

## Roadmap

The high-level roadmap for 2026 includes:
1. Launching the website
2. Python Basics for Everybody
3. Linux Basics for Hackers
4. Development of the club creation platform
5. Advanced Python and Django
6. Security fundamentals: how to not get hacked

Details and progress are displayed on the live roadmap page.
